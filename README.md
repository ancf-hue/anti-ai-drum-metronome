## 📱 О проекте

**«Анти ИИ Барабан»** — это мобильное веб-приложение, сочетающее в себе **метроном** и **тренажёр ритма**. Проект создан для развития чувства ритма, музыкального слуха и координации.

Приложение работает полностью в браузере, не требует установки, поддерживает установку на экран телефона как PWA и работает офлайн после первого открытия.

## ✨ Возможности

| Функция | Описание |
|---------|----------|
| 🎵 **Метроном** | Настройка BPM от 40 до 200 с визуальным бегунком |
| 🥁 **Барабанная подушка** | 6 различных звуков (Kick, Snare, Hi-Hat, Cowbell, Woodblock, Laser) |

| 🌓 **Светлая/Тёмная тема** | Выбор оформления с сохранением в памяти |
| 📱 **Адаптивный дизайн** | Экран 9:16, оптимизирован для мобильных устройств |
| 📲 **PWA поддержка** | Можно установить на главный экран телефона |

## 🎮 Как играть

1. **Запустите метроном** — нажмите кнопку «СТАРТ»
2. **Слушайте ритм** — метроном тикает с заданной скоростью BPM
3. **Отбивайте ритм** — нажимайте на круглый барабан или клавишу **Пробел** на клавиатуре
4. **Получайте оценку** — чем точнее попадание, тем выше балл
5. **Улучшайте рекорд** — старайтесь достичь 100% точности

### Шкала точности

| Отклонение | Оценка | Очки |
|------------|--------|------|
| < 30 мс | 🎯 ИДЕАЛЬНО! | 100 |
| < 70 мс | 👍 ХОРОШО | 70 |
| < 150 мс | ⚡ ПОПАЛ | 40 |
| > 150 мс | 😅 МИМО | 0 |
